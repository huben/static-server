### usage

```
$ static-server --help 

USAGE:
    static-server [OPTIONS]

OPTIONS:
    -h, --host <HOST>    [default: 127.0.0.1]
        --help           Print help information
    -o, --open
    -p, --port <PORT>    [default: 3030]
    -r, --root <ROOT>    [default: .]
    -V, --version        Print version information
```
